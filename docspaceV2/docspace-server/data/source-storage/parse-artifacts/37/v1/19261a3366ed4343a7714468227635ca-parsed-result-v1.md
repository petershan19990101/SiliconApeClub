# 引言

## 概述

Switch集成信贷路由系统接入规范定义了接口开发规范、通讯规范以及通信接口的定义。由于Switch集成信贷路由系统为对接上游订单系统和下游资金系统，本规范主要用于Switch转换路由以及助助贷业务流程。

## 适用范围

本文档的适用对象为系统技术开发人员、日常维护人员以及需求人员。他们需要熟悉以下基本知识：

- 熟悉贷款业务流程，了解其基本原理。

- 熟悉dubbo框架远程调用方式(非需求人员)。

- 掌握业务防重/幂等操作。

## 数据规范

- 系统之间数据传输以dubbo协议进行交互。

- 若业务系统请求数据为文件格式（图片、PDF等），核心系统提供具有读写权限的文件存储地址，业务系统上传文件后将上传文件的全路径同请求参数一并发送给联合贷款前置系统。

- 日期字段类型均采用yyyyMMdd格式，时间字段均采用yyyyMMddHHmmss。

# 接口规范

文档出现”+”时标识内部层级结构，需要与之前最近对象结构作为整体进行理解。

必填列说明：

- Y - 任何情况不允许为空

- N - 任何情况下可以为空

- C - 条件允许为空，具体看说明列
